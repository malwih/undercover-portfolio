UNDERCOVER SOCIETY - Roblox Map Portfolio

Cara pakai di GitHub Pages:
1. Upload semua isi folder ini ke repository GitHub.
2. Masuk ke Settings > Pages.
3. Source pilih Deploy from a branch.
4. Branch pilih main / root.
5. Buka link GitHub Pages yang muncul.

Ganti foto profil:
- Masukkan foto kamu ke folder: assets/profile/
- Nama file wajib: profile.jpg
- Kalau mau pakai PNG, ubah src di index.html dari profile.jpg menjadi profile.png.

Ganti slide foto:
- Masukkan 4 foto ke folder: assets/slides/
- Nama file wajib:
  slide-1.jpg
  slide-2.jpg
  slide-3.jpg
  slide-4.jpg
- Website otomatis crop semua foto ke rasio 4:5 menggunakan object-fit: cover.
- Foto tidak diperkecil/letterbox, tapi di-crop agar penuh.

Ganti musik:
- Masukkan musik ke folder: assets/music/
- Nama file wajib: background.mp3
- Catatan: beberapa browser memblokir autoplay bersuara. Website sudah dibuat untuk mencoba autoplay, lalu otomatis mengaktifkan musik saat user pertama kali klik/tap halaman.
